It may take a minute to load all the graphs. Please be patient.

In order to view the graphs, you need to run this page on a local server. One easy was to set up the local server is by using npm.

  1. Install Node and npm: npm install npm@latest -g    instructions
  2. Install http-server : npm install http-server -g    instructions
  3. Start the server: http-server

The graphs depict the number of good lines versus buggy lines for each developer in the repository. Also Average Daily Ratio and the overall ratio is added below each graph.
